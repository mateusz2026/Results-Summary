# Results Summary Component

## 📌 Overview
This project is a solution to the Frontend Mentor Results Summary Component challenge.  
The goal was to build a responsive component that displays results and summary data.

## 🚀 Live Demo
https://mateusz2026.github.io/Results-Summary/

## 🛠️ Built with
- HTML5
- CSS3
- Flexbox
- Mobile-first workflow

## 📱 Features
- Responsive design (mobile & desktop)
- Clean UI layout
- Summary sections with scores
- Styled button with hover effect

## 📚 What I learned
- How to structure layout using Flexbox
- How to align elements properly
- Creating responsive components
- Styling UI elements like cards and buttons
